This repository contains Python assignments and practice exercises covering:

* Conditional statements

The goal of this project is to practice Python programming skills and apply them in real examples.

---

## **How to Run the Code**

1. Make sure you have Python installed.
2. Download or clone the repository:

   ```bash
   git clone https://github.com/Ishnaome/naomi_python_assignment.git
   ```
3. Navigate to the project folder:

   ```bash
   cd naomi_python_assignment
   ```
4. Run any Python file:

   ```bash
   python filename.py
   ```
